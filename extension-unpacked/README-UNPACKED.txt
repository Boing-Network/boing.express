Boing Express — unpacked extension (pre-review / local testing)
===============================================================

This folder is generated by:  pnpm run build:extension:unpacked
Do not edit files here; they are overwritten on each build.

Load in Chrome
----------------
1. Open chrome://extensions
2. Turn on "Developer mode"
3. Click "Load unpacked"
4. Choose this folder: C:\Users\chiku\Desktop\vibe-code\boing.express\extension-unpacked

Submitting to Chrome Web Store
-------------------------------
Use the zip workflow instead (bundles the same files):
  pnpm run zip:extension
Produces: boing-wallet-extension.zip at the repo root.

Relative path from repo root: extension-unpacked/
